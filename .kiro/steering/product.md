# Product Overview

若年者ものづくり競技大会（業務用IT職種）の本番課題を練習・実装するためのリポジトリ。EC受注管理デスクなど業務アプリの仕様策定から実装までを反復する基盤を提供。

## Core Capabilities

- 仕様駆動（AI-DLC/Kiro）で要件→設計→タスク→実装を段階管理
- Windows/.NET + SQL Server を前提とした業務アプリ開発の演習
- データベース設計とデスクトップUIフローの一体検証を想定
- 競技要件を再現した環境セットアップ手順の共有

## Target Use Cases

- 競技課題に沿った業務アプリのプロトタイプ作成
- SQL Serverを用いたデータモデル/ストアドの設計検証
- 仕様書・タスク分解の練習とレビュー循環

## Value Proposition

- フェーズ分離されたワークフローにより再現性のある開発手順を確保
- オフラインで完結する .NET + SQL Server 開発環境を短時間で再現
- 競技向けに必要十分な機能セットにフォーカスし、過度なスコープ拡大を防止
